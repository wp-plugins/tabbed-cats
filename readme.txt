== Description ==

Sometimes you need to show your category and the inside posts in a tab view. Thats where tabbed cats come in. Usage of this plugin is very simple

just put in the [tabbed_cats top="0"]  where top is the parent category so zero is all top categories. 



== Installation ==

1. Upload 'tabbed-cats' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Put in above mentioned shortcode
